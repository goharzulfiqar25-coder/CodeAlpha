# 🤖 CodeAlpha FAQ Chatbot

## 📌 Project Overview

This project was developed as part of the **CodeAlpha AI Internship - Task 2**.

The FAQ Chatbot is an intelligent question-answering system that helps users get instant answers to frequently asked questions related to the CodeAlpha AI Internship. The chatbot uses Natural Language Processing (NLP) techniques to understand user queries and find the most relevant answer from a predefined knowledge base.

The project supports both:

- Command Line Interface (CLI)
- Graphical User Interface (GUI) using Tkinter

## 💡 How It Works

1. User enters a question.
2. The chatbot preprocesses the text.
3. TF-IDF Vectorizer converts text into numerical vectors.
4. Cosine Similarity compares the user query with stored FAQs.
5. The most relevant answer is returned to the user.
6. Confidence score is displayed for transparency.
